# EmojiKeyBoard-demo
  实现了一个在App中常用到的聊天键盘的demo，表情用plist存储,格式为:[em_1]，可以添加自己表情，方便扩展。还有拍照和获取相册照片的功能，不同功能方便切换，后续会更新语音输入以及完全实现聊天气泡的封装。
